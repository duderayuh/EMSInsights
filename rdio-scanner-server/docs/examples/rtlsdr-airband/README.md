## Rdio Scanner's configuration

In you Rdio Scanner dirwatch configuration, use those parameters:

- Directory: `/tmp/radio/Laurentides`
- Extension: `mp3`
- Type: `Default`
- System: _choose the appropriate one_
- Mask: `laurentides_#DATE_#TIME_#TGHZ`
